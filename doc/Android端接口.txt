注册新用户：
user/register
POST
参数：
	device_id：String
	tel：String
	education：int
	gender：int gender,
	birthdate：String birthdate -->格式：1991-01-11
	income：int
其中的int为枚举值，具体参考
https://github.com/sjtudclab/freeWifi/tree/master/server/src/main/java/cn/edu/sjtu/dclab/freewifi/constants
返回：json
{ code：0/-1   0表示成功 }

wifi连接成功后，通知服务器：
user/notification
POST
参数：
device_id：String
wifi_id：String
返回：json
{ code：0/-1   0表示成功 }

获取wifi列表：
wifi/get
GET
longitude
latitude

返回：json
{
code：0/-1
size:
data: wifilist
}

客户端收到推动的广告后，点击notification会进入广告页面，广告的url为：domain/ad/mobile/{id}

用户每次点击广告后，会通知服务，服务器收集点击数据，便于进行数据分析：
user/click
POST
参数：
device_id：String
ad_id：String
返回：json
{ code：0/-1   0表示成功 }

用户点击收藏广告
/ad/collect/add
POST
参数：
device_id：String
ad_id：String
返回：json
{ code：0/-1   0表示成功 }

device_id标识的用户获取所有收藏的广告
/ad/collect/{device_id}
GET
返回：json
{ code：0/-1   0表示成功
size:
data: adlist }
